3. 循环控制保留字：
    - break: 跳出并结束当前整个循环，执行循环后的语句
    - continue:结束当此循环，继续执行后续次数循环

4. 循环与else:
    当循环没有被break语句退出时，执行else语句块，
    else语句块作为“正常”完成循环的奖励
    for <循环变量> in <遍历结构>:
        <语句块1>
    else:
        <语句块2>
        
    while <条件> :
        <语句块1>
    else :
        <语句块2>
